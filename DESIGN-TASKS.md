# TofSync — צ'קליסט יישום מסמך עיצוב/פונקציונליות v2 (עם מידות מדויקות)

מקור: **מסמך הנחיות עיצוב TofSync v2** (14 עמ', כולל מידות px מדויקות לכל רכיב) -
מחליף את גרסת v1 ששימשה ל-Phase 0.
עבודה לפי שלבים - כל Phase = שיחה נפרדת (להריץ `/clear` ביניהן), commit בסוף כל Phase.
מספור הסעיפים בקובץ הזה (Phase 0-12) **נשאר כמו ב-v1** (לפי סדר עדיפויות סעיף 19/20),
אבל כל Phase מצוין עכשיו עם המידות המדויקות מ-v2 כדי שלא נצטרך לנחש כמו ב-Phase 0.
